# EasyEmu

A unified interface for playing emulated games across all gaming systems.

## Features

- **Unified Library** - All your games in one place with automatic ROM detection
- **Tons of Platforms** - NES, SNES, PlayStation, GameCube, and many more
- **Auto Metadata** - Fetches cover art and game info automatically
- **Controller Support** - Navigate the entire app with your gamepad
- **Built-in Emulation** - Play directly or launch external emulators
- **Save Management** - Save states and battery saves handled automatically

## Quick Start

1. Download and run EasyEmu
2. Add your ROM folders in the setup wizard
3. Let EasyEmu detect your installed emulators
4. Click any game to play

See the full [Quick Start Guide](QUICKSTART.md) for detailed instructions.
